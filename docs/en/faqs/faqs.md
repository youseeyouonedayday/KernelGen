# FAQs

---

**Q: How long is the KernelGen Operator Development MCP Toolkit token valid?**

A: The token is currently valid for one year. If it expires, you will need to renew it on the official website.

---

**Q: What happens when the KernelGen Operator Development MCP Toolkit token expires?**

A: An error message such as `[MCP] Error activating server code` will appear on the MCP settings page. You will need to renew the expired token on the official website.

---

**Q: Which chips does KernelGen Operator Development MCP Toolkit support?**

A: It currently supports common mainstream chips, including NVIDIA, Huawei Ascend, Iluvatar CoreX, Hygon, Moore Threads, and MetaX.

---

**Q: Can I use KernelGen Operator Development MCP Toolkit in scenarios involving sensitive data?**

A: Yes. KernelGen Operator Development MCP Toolkit protects sensitive data through masking/desensitization and hash-based desensitization, ensuring that sensitive data is not leaked.
