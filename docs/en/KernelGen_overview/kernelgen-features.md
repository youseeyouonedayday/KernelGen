# Features

KernelGen Web Platform and KernelGen Operator Development MCP Toolkit platforms support generating, optimizing, and specializing operators through one of the following methods:

- KernelGen Web Platform
- KernelGen Operator Development MCP Toolkit and its MCP tools: All MCP tools are integrated into the MCP server.
- KernelGen Operator Development MCP Toolkit and KernelGen skills: All KernelGen skills are bundled in the kernelgen-flagos skill. Users can invoke the bundled skill to invoke the sub-skills.
