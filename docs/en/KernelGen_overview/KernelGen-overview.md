# KernelGen Overview

**KernelGen** is designed to generate, optimize, and specialize operators across diverse hardware platforms. After connecting to KernelGen Operator Development MCP Toolkit, you can either accomplish these tasks using the server alone or combine it with first- and third-party skills. Users can invoke KernelGen Operator Development MCP Toolkit and these skills in VS Code (and GitHub Copilot), Claude Code, or OpenClaw.
