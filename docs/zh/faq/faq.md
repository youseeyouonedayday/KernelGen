# 常见问题解答

---

**Q: KernelGen MCP Server 的 token 有效期是多久？**

A: 目前 token 有效期是一年，如果 token 失效，需要在官网续期

---

**Q: KernelGen MCP Server 的 token 过期了会发生什么？**

A: token 过期会在 MCP 设置页面显示报错信息诸如："[MCP] Error activating server code"。token 过期需要在官网续期。

---

**Q: KernelGen MCP Server 支持哪些芯片？**

A: 目前支持常见主流芯片如 Nvidia，华为昇腾、天数、海光、摩尔线程、沐曦股份等。

---

**Q: 我可以在涉及敏感数据的场景中使用 KernelGen MCP Server 吗？**
A: 可以。KernelGen MCP Server 针对敏感数据进行遮盖脱敏和哈希脱敏保护，可以保护敏感数据不会泄露。
