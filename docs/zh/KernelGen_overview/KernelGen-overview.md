# KernelGen 概述

**KernelGen** 旨在跨多种硬件平台生成、优化和特化算子。连接 KernelGen MCP 服务器后，您可以单独使用该服务器完成上述任务，也可以将其与第一方和第三方技能（Skills）结合使用。用户可在 VS Code（及 GitHub Copilot）、Claude Code 或 OpenClaw 中调用 KernelGen MCP Server 和这些技能。
